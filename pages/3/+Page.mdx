# Extensions

For example, UI framework (React/Vue/Solid/...) integration:

- vike-react &lt;=> Next.js/Remix
- vike-vue &lt;=> Nuxt
- vike-solid &lt;=> Solid Start

Ejectable: you can, if you want, fully integrate React/Vue/Solid  
yourself instead.

- Quickly get started with `vike-{react,vue,solid}`
- Later, for extra flexibility, you can eject
